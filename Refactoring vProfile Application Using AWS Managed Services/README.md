## Overview
This project demonstrates refactoring a monolithic Java-based application (**vProfile**) into a **scalable, highly available CloudOps architecture** using AWS managed services.

The focus is on **operating and managing production workloads** with scalability, reliability, security, and observability.

---

## Architecture Summary
- Application hosted on **Elastic Beanstalk**
- Load balanced using **Application Load Balancer**
- Auto scaling enabled for high availability
- Backend services decoupled using **RDS, ElastiCache, and Amazon MQ**
- Global content delivery using **CloudFront with SSL**
- DNS management via **Route53**
- Centralized monitoring and logging using **CloudWatch**
- Secure access control using **IAM roles**

---

## Technologies Used
Elastic Beanstalk, EC2, ALB, Auto Scaling, RDS, ElastiCache, Amazon MQ, S3, CloudFront, Route53, CloudWatch, IAM, Apache Tomcat

---

## Key CloudOps Highlights
- Production-ready deployment
- High availability (Multi-AZ)
- Managed AWS services to reduce operational overhead
- Monitoring, logging, and health checks
- Secure IAM-based access control
